# Rafeeq

Rafeeq is an immersive, full-screen AI assistant for the terminal. Built for the LOS Hackathon, it acts as an extension of your mind, helping you capture notes, manage tasks, and stay organized through natural conversation.

## Core Philosophy
- **TUI-First:** A beautiful, full-screen terminal experience using the `Textual` framework.
- **Agentic Engineering:** Built entirely by AI agents using iterative research and execution.
- **Privacy-Centric:** All data is stored locally in simple JSON or SQLite.

## Features
- **Conversational Capture:** Save thoughts and ideas as naturally as sending a text.
- **Intelligent Task Management:** Track, update, and complete tasks with ease.
- **Daily Briefing:** Get a clear overview of your day and recent notes.

## Tech Stack
- **Python**
- **Textual** & **Rich**
- **Gemini API**
- **SQLite/JSON**
